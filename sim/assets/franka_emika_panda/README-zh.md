# Franka Emika Panda 机器人模型（MJCF，中文）

> 该模型基于 MuJoCo Menagerie 提供的 Franka Panda 资产，适用于 MuJoCo 场景加载与控制实验。

## 内容概览

本目录主要包含：

- `panda.xml`：机械臂与夹爪主体模型
- `scene.xml`：基础场景（地面、光照、可视化设置）
- `assets/`：视觉与碰撞网格
- `CHANGELOG.md`：变更记录
