  - `tests/test_mujoco_gripper_driver.py`

## Runtime Interface

`run_mujoco_panda_gripper.py` supports repeated console commands during simulation:

- `open`
- `close`
- `set <0-255>`
- `width <0-255>`
- `gripper open|close|set <value>`
- `quit`
